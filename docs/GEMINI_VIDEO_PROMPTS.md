## üìù Tips for Using These Prompts

1. **Iterate**: Generate multiple versions, pick the best
2. **Consistency**: Use similar style descriptors across prompts
3. **Specificity**: Add more details if results aren't quite right
4. **Combine**: Mix and match elements from different results
5. **Refine**: Use "make it more [adjective]" to adjust
6. **Aspect Ratios**: Specify 16:9 for video, 9:16 for vertical
7. **Style References**: Add "in the style of [brand/artist]" if helpful
