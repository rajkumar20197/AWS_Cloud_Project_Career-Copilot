## 30-Minute Quick Version

If you need a demo video FAST, follow this streamlined process:

### ‚úÖ Step 1: Record Screen (10 minutes)

1. **Open your app** at `http://localhost:3000`
2. **Use OBS or Loom** to record
3. **Record these screens:**
   - Landing page (5 seconds)
   - Dashboard (5 seconds)
   - Job Search with AI scores (10 seconds)
   - Resume Optimizer (5 seconds)
   - Career Roadmap (5 seconds)
   - Market Intelligence (5 seconds)

**Total footage needed:** ~35 seconds of clean screen recordings

### ‚úÖ Step 2: Generate Visuals (10 minutes)

Use Gemini/DALL-E with these quick prompts:

1. **Stressed student:**

   ```
   Stressed college student at laptop, job search, tired, photorealistic
   ```

2. **Happy student:**

   ```
   Happy college student celebrating job offer, laptop, photorealistic
   ```

3. **Logo/CTA screen:**
   ```
   Clean tech logo design, "AI Career Agent", blue purple gradient, minimalist
   ```

### ‚úÖ Step 3: Edit Video (10 minutes)

**Use CapCut (easiest):**

1. Import all clips
2. Arrange in order:
   - Stressed student (3 sec)
   - Screen recordings (35 sec)
   - Happy student (3 sec)
   - Logo/CTA (4 sec)
3. Add text overlays:
   - "AI Career Agent"
   - "Automated Job Search"
   - "Get Started Free"
4. Add free music from YouTube Audio Library
5. Export as MP4 (1080p)

**Done!** You have a 45-second demo video.
